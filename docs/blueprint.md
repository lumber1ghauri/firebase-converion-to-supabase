# **App Name**: GlamBook Pro

## Core Features:

- Service Selection: Allow users to select makeup services (Bridal, Semi Bridal, Party, Photoshoot) for specific days.
- Multi-Day Booking: Enable users to book services for multiple consecutive days.
- Contact Details Requirement: Require users to enter their contact details before displaying the pricing breakdown and complete quote.
- Location Selection: Allow users to select 'Toronto' or 'Outside Toronto' to adjust pricing accordingly.
- Automated Quote Generation: Automatically calculate and display the total cost based on selected services, days, and location.
- Email Automation: Automatically send a detailed email to the user with the generated quote and booking details, to be reviewed and processed later.
- Responsive Design: Ensure the application is fully responsive and provides a seamless experience across all devices.
- Intelligent Availability: Use AI as a tool to intelligently update your availability based on bookings, travel time in the Toronto GTA, and service duration using a map/routing tool.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) for elegance and sophistication, reflecting high-end beauty services.
- Background color: Light gray (#F5F5F5), a desaturated version of the primary hue for a clean and modern backdrop.
- Accent color: Rose pink (#E91E63) to highlight key actions and important information, analogous to purple on the color wheel.
- Font pairing: 'Belleza' (sans-serif) for headlines, 'Alegreya' (serif) for body text. Note: currently only Google Fonts are supported.
- Use elegant, minimalist icons to represent different makeup services and booking options.
- Clean and intuitive layout with a focus on visual appeal to showcase the makeup artist's expertise.
- Subtle transitions and animations to enhance user interaction and provide a smooth booking experience.
